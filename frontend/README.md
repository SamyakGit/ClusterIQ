# ClusterIQ Frontend

React-based dashboard for visualizing Databricks optimization recommendations.

## Features

- **Real-time Dashboard:** Monitor jobs, clusters, and recommendations
- **AI Recommendations:** View AI-generated cost optimization suggestions
- **Auto-refresh:** Configurable real-time updates
- **Modern UI:** Built with React, Tailwind CSS, and Vite

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- React Query (TanStack Query)
- Axios
- React Router
- Lucide Icons

## Running

```bash
npm install
npm run dev
```

## Building

```bash
npm run build
```

The built files will be in the `dist` directory.

## Environment Variables

Create a `.env` file:
```
VITE_API_URL=http://localhost:8000
```
